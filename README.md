🧩 Гра «П’ятнашки» (15 Puzzle)

Це комп'ютерна реалізація класичної настільної головоломки «П’ятнашки», розроблена в рамках комплексної практичної роботи. Проєкт дозволяє гравцеві випробувати свої логічні навички на полях різного розміру.
📝 Опис проєкту

«П’ятнашки» — це популярна головоломка, що складається з квадратного ігрового поля та набору пронумерованих фішок. Одна комірка на полі завжди залишається порожньою, що дає можливість гравцеві переміщати сусідні фішки.

Мета гри: упорядкувати фішки за номерами (від 1 до 15 або більше, залежно від розміру поля) зліва направо та зверху вниз, залишивши порожнє місце у нижньому правому куті.
🚀 Основні можливості

    Налаштовуваний розмір поля: Користувач може обрати складність гри при запуску. Допустимий розмір поля — від 3x3 до 9x9.

    Випадкова генерація: На початку гри фішки розташовуються у довільному (перемішаному) порядку.

    Інтуїтивне керування: Переміщення фішок здійснюється за правилами класичної гри (лише по горизонталі або вертикалі в порожню комірку).

    Перевірка виграшу: Система автоматично визначає момент, коли всі фішки стали на свої місця.

📏 Правила гри

    Хід: Натисніть на фішку, що межує з порожньою коміркою, щоб перемістити її на вільне місце.

    Обмеження: * Можна пересувати лише одну фішку за один хід.

        Пересування можливе лише по вертикалі або горизонталі.

        Заборонено пересувати фішки по діагоналі або міняти їх місцями вручну.

    Перемога: Гра вважається завершеною, коли досягнуто цільового стану (цифри від 1 до N розташовані послідовно).
